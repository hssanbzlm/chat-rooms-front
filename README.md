# *Chat-rooms (frontend)*
## *Features*
- Create an account 
- Create a room 
- Join a room 
- Chat in public rooms 
- Chat privately with users of the same room 
- Search connected users
- Get typing status
- Update profile (avatar and full name)
- Destroy a room
